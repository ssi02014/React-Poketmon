## 🔖 Pokemon App을 사용하기 전 행동 수칙
1. **" npm install or yarn install "** 을 clone한 폴더에 해주세요. (프론트엔드 종속성 다운)
2. clone 한 폴더에서 **" npm start or yarn start "** 를 통해 App을 실행시켜주세요.
3. App 기능을 확인합니다.
